基于Python Flask定时通过邮件发送天气预报。天气数据来自高德天气查询API，参考https://lbs.amap.com/api/webservice/guide/api/weatherinfo。
当前通过 Python APScheduler 模块实现 JOB 任务。
每日天气发送
每天通过邮件发送：
今日温度
体感温度
和昨天相比
降雨
需要带伞，穿衣推荐
…..
